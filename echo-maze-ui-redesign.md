# Echo Maze UI 修改方案

## 设计约束

- **不改**：整体布局结构（导航栏+左侧面板+CRT监视器+右侧面板+底部栏）
- **不改**：CRT扫描线、暗角、色差文字、零emoji、手绘像素
- **不改**：字体系统（等宽终端字体+系统UI字体）
- **要改**：**语义重命名**——一切标签从"迷宫/回声行者/静默"转向"设施/工程师/声场事故"
- **新增**：聚焦脉冲充电环、护盾指示器、叙事碎片面板、终章选择界面

---

## 一、顶栏导航

### 当前
```
ECHO_MAZE v4.0    [HELP] [中/EN] [SND:ON]
```

### 修改后
```
CADMUS STATION v0.XXX    [SYS] [LOG] [中/EN] [SND:ON]
```

**变更明细**：
- 品牌名从 `ECHO_MAZE` 改为 `CADMUS STATION`（卡德摩斯站）
- 版本号改为 `v0.XXX`（遵循用户规则）
- HELP按钮 → SYS（系统信息面板，按H打开设施地图+操作手册）
- 去掉无用的 HELP 独立按钮，整合到SYS中
- MUTE / SND 保持

**Logo/图标**：旧版logo图标改为像素化的设施徽章（圆形齿轮+声波弧线图案），16×16像素手绘。

## 二、进场画面（Splash）

### 当前核心元素
- 标题：回 声 迷 宫 / ECHO MAZE
- 标语：你看不到迷宫——你只能"听"见它
- DIFFICULTY / MAP / THEME 选项
- ECHO WALKER 角色选择
- MODIFIERS 词缀
- START / DAILY / RANK 按钮

### 修改后

#### 标题
```
卡 德 摩 斯 站
CADMUS SUBSONICS STATION
```

保留色差文字特效（品红+青色偏移），但整体色调从绿/霓虹改为**蓝白**调（对应"声场/声纳"概念）。

#### 标语
```
"声纳系统正常。电池剩余：10。目标是关闭核心，找到幸存者。"
```

字体像素风格，暗蓝色文字，以打字机效果逐字显示（模拟外骨骼启动画面）。

#### 选项区（保留位置，内容变更）

| 行 | 当前 | 修改后 |
|---|------|--------|
| 第1行 | DIFFICULTY | FACILITY_CLEARANCE（设施权限级别）—— EASY/STD/HARD → LEVEL_A / LEVEL_B / LEVEL_C |
| 第2行 | MAP | SECTOR（扇区选择）—— 保留网格大小，命名改为"扇区规模" |
| 第3行 | THEME色板 | 保持，但色板扩展为6种"设施模式"：标准/应急/消光/深岩/安保/核心，每关自动切换 |

#### 角色选择（完全重写）

旧："ECHO WALKER" + 六个角色卡（先知/幽灵/巨兽/歌者/暗影/共振者）
新："EXOSKELETON MODEL" + 标准外骨骼（默认）+ 六个可解锁外骨骼

角色卡设计：
- 每一张卡片=一个**外骨骼型号名称**+**对应人名称**+**解锁条件**
- 已解锁：蓝色边框+外骨骼像素图标+状态"ACTIVE"
- 未解锁：灰色+问号像素图标+解锁条件提示

```
┌─────────────────────────────────────────┐
│  STANDARD MK-I     林先和的遗物         │
│  [像素外骨骼图]    [像素外骨骼图]       │
│  ACTIVE            需击败回响之母       │
└─────────────────────────────────────────┘
```

卡片6张排列为3×2网格，和原来一样。

#### 词缀（保留但减少）

保留：SILENT / HUNTED（移除其他四个）。标签名改为中文+英文。

#### 底部按钮

移除：DAILY / RANK
修改：START 文案改为"DEPLOY >"（部署）

## 三、游戏内——左侧面板

### 当前结构
```
ECHO_WALKER   [角色图标+名称]
SIG_PWR       [能量条+数值]
STATS         [脉冲/步数/分数/关卡]
SLNT_         [连击数]
AUG           [能力标签]
MODS          [物品]
MSG           [消息]
```

### 修改后

```
EXO_STATUS        [外骨骼型号+装配人名字]
░░░░░░░░░░░░░░    [能量条10格，分两段显示]
├ 探测脉冲 · 1    [左对齐灰色]
└ 聚焦脉冲 · 3    [左对齐蓝色，蓄力时亮起]
────────────────
SCAN_RECORD       [脉冲次数/步数/关卡]
SIGNAL            [碎片进度 X/18]
LOG               [最新一条扫描结果]
```

**变更明细**：

**EXO_STATUS 区块**
- 标题从 `ECHO_WALKER` 改为 `EXO_STATUS`
- 显示：外骨骼型号名（例："STANDARD MK-I"）+ 装配人标识（例："未指定"或"林先和的遗物"）
- 移除大图标，压缩为一行文字

**能量系统 区块**（全新建）
- 标题：`BATT`（电池）
- 能量条保持10格绿色渐变的视觉
- 能量条下方新增两行消耗指示：
  ```
  | SCAN · 1                    |
  | FOCUS · 3  [蓄力中...]      |  ← 长按空格时，"FOCUS · 3"变成高亮蓝
  ```
- 当玩家开始长按空格蓄力时，"FOCUS · 3"右侧出现计时环（文本提示："CHARGING ▮▮▮"），满0.5秒后释放→发射聚焦脉冲
- 能量低于3时，"FOCUS · 3"变成暗红色，提示能量不足

**SCAN_RECORD 区块**（原STATS）
- 保持脉冲/步数/关卡三项
- 移除"分数"（分数移到右侧面板RATING）
- 新增一行：`CLR 许可` → 当前使用的通行权限卡数量（钥匙计数）
- 标题从 `STATS` 改为 `SCAN_RECORD`

**SIGNAL 区块**（全新建，替代原SLNT_连击）
- 标题：`SIGNAL`（声纳信号记录）
- 显示：当前收集的叙事碎片数量 X/18
- 视觉：18个小格子排列成3行×6列。已收集=蓝色填充，未收集=灰色轮廓。每收集一个新碎片，对应格子播放一次亮起动画
- 下方一行：新碎片数量 `NEW: +X`
- 碎片暂停屏（按TAB打开）会在游戏暂停时显示

**LOG 区块**（原MSG）
- 标题从 `MSG` 改为 `LOG`
- 内容从"通用消息"改为"外骨骼扫描输出"：
  - 进入新关：显示"扇区扫描完成——检测到X个结构异常"
  - 找到碎片：显示"声学记录回放——[人物名]·片段[编号]"
  - Boss出现：显示"警告——强回声源检测"
  - 低能量：显示"电池警告——剩余能量<3"

## 四、游戏内——CRT中央监视器

### 当前
- 画布渲染像素画面
- 扫描线+暗角覆盖
- 右下角绿色LED

### 新增

**聚焦脉冲充电环**（屏幕中央，仅蓄力时显示）
- 长按空格>0.2秒时，屏幕中央出现一个蓝色收缩环（环形像素动画）：外圈→内圈逐渐收缩，0.5秒时到达最小直径→环变为亮白色→释放
- 视觉参考：声纳屏幕上的"ping圆环"，但从外向内缩小
- 如果提前释放（<0.5秒），环直接消失，无视觉效果
- 环的颜色映射当前能量：蓝（充足）→黄（中等）→红（低）

**Boss护盾指示器**（Boss战专属，屏幕上方居中）
- 三个护盾格子：`[▓] [▓] [▓]`，横向排列，像素边框
- 每摧毁一层 → 对应格子碎裂动画（像素碎片散落）→ 变暗
- Boss名字下方一行文字，随护盾减少而变化：
  - 3层："声场稳定"
  - 2层："频率波动"
  - 1层："结构崩解"
  - 0层："核心暴露——最后一击"

**关卡叙事文字**（每关开始时，屏幕中央，持续3秒后淡出）
- 第一行：关卡名称（例："第3关：声场模拟室"）
- 第二行：设施位置（例："CADMUS-03-REACTOR"）
- 第三行：外骨骼扫描结果（例："检测到强回声干扰源"）
- 文字依次出现（打字机效果），3秒后整体淡出
- 字体：像素等宽，白色，有蓝绿色辉光

**通行权限卡拾取提示**（屏幕左下角，类似事件提示）
- 当玩家靠近钥匙时，左下角出现一行闪烁文字："权限卡检测——50cm"

## 八、终章选择界面（全新建）

### 触发
第10关，进入控制室，吴暗对话结束后。

### 设计
- 中央弹出，类似modal但有所不同——三道竖排选项，分别从屏幕左中右滑入
- 背景：核心控制室的CRT画面（不动），六人的回声印记在屏幕边缘发光

```
┌────────────────────────────────────────────┐
│                                            │
│    [A]                [B]                  │
│  配合吴暗            揭露真相              │
│  关闭核心            独自关闭              │
│  一起出去            让他留下              │
│                                            │
│   解锁：暗影          解锁：共振者          │
│                                            │
│              [全碎片已收集]                 │
│                 [C]                        │
│              炸毁设施                       │
│              一切结束                       │
│              解锁：歌者                     │
│                                            │
│          按 A / B / C 选择                  │
└────────────────────────────────────────────┘
```

A和B始终可见。C在全18块碎片收集后才会出现（从下方滑入，有金色辉光）。

- 玩家确认选择→其他选项淡出→选中选项居中放大→屏幕闪白→终章文字逐行显示
