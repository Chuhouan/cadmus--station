# Echo Maze 叙事剧本 v5.0

## 主角设定

你是卡德摩斯站的应急工程师。四十二岁。做这行十七年了。你修过比这个更大、更深、更危险的东西。但你没有在完全看不见东西的情况下修过任何东西。

你不是自愿下来的。两周前，你在地面上接到调令——"卡德摩斯站核心异常，需要声纳外骨骼操作员。"那就只有你了。那套旧型号的外骨骼是你十年前参与设计的。十年前的东西。但你记得它的脾气。

你不知道那六个人。名单你看过。林先和，首席工程师——你在行业期刊上见过他的名字。其他的，苏无影、赵铁山、陈歌、吴暗、郑振，你只知道名字和职位。六个人。三天了。

你告诉自己：下去，找到他们，关闭核心，爬出来。

你早知道不会那么简单。这行就这样。

## 二、十关叙事线

### 第1关：应急通道

**进场**：
```
我发射了第一次脉冲。墙壁在屏幕上亮了三秒，然后慢慢暗下去。像有人举着火柴走过一个没有窗的房间。这套外骨骼的声音和十年前一样——高频率的嗡鸣，然后安静。

声纳显示前方三十米处的闸门锁死了。控制面板需要一块备用电池。我记得这个设计。十年前我们加这个锁，是为了防止有人在声场测试期间误入。现在它锁住了六个人退出来的路。

也许他们还活着。也许没有。
```

**清关后**：
```
闸门开了。我往里走了一步，然后站住。我回头看了一次——应急通道往上五十米就是地面。天还是亮的。

我继续往下走。
```

### 第2关：控制室废墟

**进场**：
```
控制室烧过。墙壁焦黑。空气里有股焦糊味，外骨骼过滤不掉。

地上有水——不知道是管道破裂还是灭火留下的。水里飘着纸片。我捡起一张。林先和的手写日志："核心频率在自行爬升。"

他写的是"自行"。他不确定是不是故障。

第一个人找到了。在墙角——不是尸体。是外骨骼自动记录的声纳残像。赵铁山在这里停留过。他的最后一帧扫描数据刻在墙上："下面有风。出口在下面。"

下面不是出口，赵铁山。下面是核心。

我还不想告诉他。
```

### 第3关：声场模拟室（Boss回响之母）

**进场**：
```
声场模拟器还开着。它应该已经停了三天了。但它在自主运行——不断发射干扰脉冲，覆盖我的扫描范围。每次我ping它，它ping回来。比我快。比我大。

这不是故障。故障会自己停下来。这东西在回答我。

林先和如果能走到这里，他会知道为什么。
```

**Boss战击败后**：
```
模拟器裂开。核心残骸里有一块数据板。

林先和的第二条记录："我找到远程登录的物理来源了。信号来自设施内部。有人还在里面。"

我坐在残骸旁边，把这句话读了三遍。

有人还在里面。

不是六个人。是七个人。第七个人不在名单上。第七个人在事故发生后还在往核心控制系统发指令。

我把数据板收进外骨骼的存储槽里。林先和，你发现的事故原因是对的。但你不知道的是——你不该让他发现你知道了。
```

**第一幕收束节拍——过关后弹出**：
```
我叫______。三个小时前我以为我是来找人的。

现在我明白了。我在找一个人。不是六个失踪者——是一个不在名单上的人。他几分钟前还活着。他可能还在里面。

林先和发现了他的账户名。林先和现在在哪？
```

### 第4关：维护竖井

**进场**：
```
竖井很深。平台上全是暗水——不是水，是某种声波冷却液。踩上去黏稠，外骨骼的液压系统在抗议。好消息是暗水里没法脉冲，捕食器听不到我。坏消息是我也看不见。

赵铁山的箭头刻在墙上。隔几步就有一个——歪的。越来越歪。他在往下爬的时候已经慌了。

每层都有岔路。每条岔路他都在选。我猜他以为自己在找出口。

他一直在往下。而每往下一层，离出口就远一点。
```

**找到赵铁山的尸体**：
```
一个死胡同。暗水池的尽头。

赵铁山的声纳残像蹲在墙角。不是恐惧的姿势。是累透了。他把外骨骼头盔摘了，放在膝盖上。墙壁上有他最后刻的字："别往下走。"

我站了很久。

你走错了，老赵。你不是方向感不好。你是太相信"往深处走一定能出去"——你是矿工，你的本能就是往下。但这下面没有矿。

我把他的头盔捡起来。外骨骼还在记录——他的最后一帧扫描显示，他死前还在找风。

别往下走。我已经在下面了。
```

### 第5关：通讯中继站

**进场**：
```
陈歌沿途放了很多中继器。每一个都是一次全屏扫描——她把自己的能量留着，用中继器的电池代替。她比我聪明。

踩到第一个中继器时，她的声音自动播放："这里是陈歌。我把中继器放在沿途了。如果你听到这个，我在C-12通道。"

她的声音很稳。不害怕。或者她把害怕藏得很好。

信号干扰太强了。我自己的脉冲发不出去。我只能靠她留下的东西看见路。

陈歌，你在C-12通道等到的，是什么？
```

**找到第三个中继器**：
```
"通道里有东西……对声音敏感。我不能再脉冲了。我会关掉外骨骼，摸黑走出去。祝我好运。"

她把外骨骼关了。

在一个完全黑暗、全是声敏捕食器、不知道出口在哪里的地方——她主动放弃了唯一能让她"看见"的东西。然后走进去。

我做不到。

我也不用做到。她给我留了中继器。
```

**清关后**：
```
陈歌的中继器用完了。出口在C-12通道的尽头。她已经不在了。通道里有无人机残骸——她摸黑走的时候肯定触发了一些。但她过去了。

我相信她过去了。没有证据。但我不需要证据。
```

### 第6关：安保控制区（Boss静默猎手）

**进场**：
```
安保通道全是无人机。它们不巡逻——它们在警戒。有什么东西把它们激活了。

核心。

墙壁上有一面完好的安保面板。储存器还在运行。我趁无人机转身时靠近，外骨骼读出了一行指令："锁定所有撤离通道。阻止任何人员离开。授权人：吴暗。"

锁定时间是事故前四小时。

四小时。他提前四个小时就知道会发生什么。

他的名字叫吴暗。他是设施监理。他锁了门。然后六个人被困在里面。

我拔出数据板。林先和，你说的"账户名：WU_AN"——不是远程黑客。是设施内部的人。是监理本人。
```

**Boss战击败后**：
```
无人机核心碎裂。我从坠毁的储存器里又读出一段数据。

实验授权书。项目名称：声场武器化测试。附带人员上限：六人。优先级：最高。

六人。不是"不超过六人做实验"。是"用六个人做实验"。

他们被计入了参数。

我的手在抖。不是因为冷。
```

**第二幕收束节拍**：
```
吴暗。设施监理。锁了门。删了文件。用了六个人的命做实验参数。

但我刚才想起来——吴暗的权限级别是代理总监。他不是决策者。

谁让他做的？

地面上的人。三天前没有来救援。他们知道。他们签了字。

我是第七个下来的人。他们让我下来，是因为他们知道核心需要两个人才能关闭。

他们不是让我来救人的。

我是来给吴暗当助手的。
```

### 第7关：档案室

**进场**：
```
档案室被涂了一层吸声材料。弹了一发脉冲——声波被吞得一干二净。这种涂层是实验品，本来是给潜艇用的。他们把原型涂在了档案室，让它变成整个设施里最安静的地方。

一个理想的——删除证据的地方。

墙会吞噬所有声音。我只能靠撞墙来感知方向——撞一下，外骨骼在那一侧记录一个触觉轮廓，两秒后消失。像在全是枕头的房间里摸索，靠碰壁来认路。

档案室里有个东西在睡觉。比普通的捕食器大三倍。蜷在房间中央，表面有红光慢慢脉动。每走一步我都在数。直觉告诉我，它听到一定步数就会醒。
```

**找到吴暗的实验文件**：
```
六份被撕碎的文件。外骨骼自动拼接。

实验授权书全文。卡德摩斯站真实目的：在战场上制造不可见区域。测试参数：满功率运行。附带人员上限：六人。

签名：吴暗。

下面还有一行小字，手写的："已获地面指挥部批准。代号：共振计划。"

我猜对了。

他们签了字。他们知道。他们派我下来的时候就知道我会发现什么。他们不在乎。他们需要的是核心被关闭——实验数据已经拿到了。

六个人的死是他们计算的代价。我是他们计算的解决方案。

我把文件存好。吴暗，你以为撕了就没了。但这套外骨骼是我设计的。它能拼任何东西。
```

### 第8关：深层岩洞

**进场**：
```
到了最底层。天然岩洞，被改造成核心降温室。这里有水晶簇——天然的巨大晶体。声波碰到它们会折射，往四个方向散开。

郑振的移动实验室在一个小舱室里。她是地质学家，本来只需要研究岩层应力。但她发现的东西比岩层重要。

她的数据终端还插着电源——不知道是从哪接的，但她接了。屏幕亮着。她的最后一封未发送邮件：

"有人从内部终端上调了功率阈值。操作员等级：设施总监。吴暗只是代理——他没有这个权限。"

地面上的人。又是你们。

她还写了一行："实验需要六个月满功率。事故后地面上没有任何回应。他们知道会发生什么。他们在等数据。"

郑振，你看懂了。

你不只是观察。你还把结论写了下来。然后你留在实验室里，继续记录，直到所有灯熄灭。
```

**清关后**：
```
岩洞里的捕食器不是威胁——它们是机会。它们巡逻时的低频嗡鸣会被水晶折射，经过回音壁反弹，一次性照亮大片区域。

我学会了用敌人的声音看东西。

郑振大概也是这么想的——不是"对抗"，是"利用"。她是地质学家，观察是她的本能。她不动，不干涉，只是把看到的东西记录下来。

有些时候，不动比动更有力量。
```

### 第9关：核心反应堆室（Boss虚空化身）

**进场**：
```
核心就在前面。十二米直径的环形发生器，在黑暗里缓慢旋转。每次旋转地面就震一下——不是声音，是纯粹的振动，透过外骨骼的液压系统传到我的骨头里。

房间里有个人形的东西。不是真人——是核心的声场投影。它在模仿六个人的外形。林先和的轮廓。苏无影的。然后模糊，变成另一个人。

房间里有三个频率锚点。只有站在锚点上发射的聚焦脉冲才能穿透投影的防御。每个锚点都在变化——我必须跟上它的节奏。

吴暗需要第七个人的原因就在这里。安全模式需要两个活人。核心投影需要一个活人去关闭。他算好了一切——除了我。
```

**Boss战击败后**：
```
核心投影解体了。最后一缕声场从林先和的轮廓散开时，我觉得我看到了他——不是真的他，是他留在核心里的声学特征。

安全模式已启动。手动刹车就在前面。

外骨骼提示：需要两个人的生物识别。

另一个活人信号——他在控制室。穿着破烂的外骨骼。他知道我来了。他需要我。

我想起赵铁山蹲在暗水池边，头盔放在膝盖上。我想起陈歌的声音，稳得没有一丝颤抖，说"祝我好运"。我想起林先和的最后一条记录——账户名：WU_AN。他写下来，放进残骸里，留给后来的人。

他们都是后来的人留给后来的人。

现在轮到我留了。
```

**第三幕收束节拍**：
```
我叫______。三个小时前我以为我是来找人的。

现在我站在核心面前。六个人留下了碎片。吴暗留下了锁。地面的人留下了签名。

我把所有碎片存进外骨骼。如果我能出去，每一条都能用。

但我不确定我想出去。
```

### 第10关：控制室（终章）

**进场**：
```
吴暗就站在刹车拉杆旁边。他的外骨骼比我旧两个型号，左腿的液压管在漏。他的脸在声纳扫描里只是一个轮廓——没有特征，没有表情。我不需要看他的脸。

"你也走到这里了。"他说。"两个人。帮我解锁。我关闭核心。我们一起出去。"

他停了。

"我的人会保证你没事。"

他说"我的人"。不是"地面"。不是"项目方"。是"我的人"。

他早就选好了。
```

**终章对峙**：
```
我叫______。这三个小时我学会了一件事。

闭嘴。听。回声会告诉你真相。

林先和的碎片开始自动播放。他在控制室发现的——"不是故障。有人在远程控制。"

苏无影的最后记录——"他杀了小陈。我把他锁在了E-3仓库。"

赵铁山的刻痕——"别往下走。"

陈歌的声音——"祝我好运。"

郑振的数据——"地面上没有来救援。他们知道会发生什么。"

吴暗在听。他的手没有从拉杆上移开。但他的轮廓在微微晃动——外骨骼捕捉到了。

"你算了一笔账。"我开口。"六个换一万。但你算错了——不是六个。是五个。小陈是你亲手杀的。不在实验参数里。你只是慌了。"

他没有说话。

"我在林先和的数据板里读到的。他在控制室发现了你的远程操作日志。小陈走过来了。你做了什么——你怕他听到。小陈只是碰巧路过。他不是实验参数。他是你害怕的时候杀的。"

"你他妈闭嘴。"声音破了。

"苏无影把你锁在E-3仓库。你杀了小陈，她锁了你。你以为她会等到死——但她死之前留下了录音。她说她没跑。她说告诉她的家人，她没跑。"

沉默。

我往前走了一步。他往后退了一步。

"你是不是觉得我会恨地面的人更多？签字的人、批准实验的人、派我下来的人？"

"我就是这么跟他们说的。"他说，"我说你会恨他们——"

"我会。但我现在站在你面前。不是在办公室前面。"

我按下外骨骼的通讯记录键。所有的碎片数据开始对外广播。地面上每一个接收器都能收到。只要有人在听。

"你锁的门。你删的文件。你杀的一个人。你牺牲的五个人。"

拉杆在我手边。

他的在我手边。

我们都没有动。
```

**三个选择**：

---

**选择A：配合吴暗**（两个活人，一起出去）

你抓住了拉杆。

他犹豫了一秒，然后抓住了另一边。你们的手在刹车杆上并排。核心开始减速——不是停下来，是停下来之前的那个嗡鸣。低沉。漫长。

"你出去后什么都不会说。"他说。不是问句。他在说服自己。

"对。"你说。"我什么都不会说。"

你们往上爬。暗水、档案室、安保区、竖井、控制室废墟、应急通道。每一步他都在你前面半格。他的右腿在发抖——液压管漏得太多了。你没扶他。

出口。天是亮的。

媒体的闪光灯你提前看到了——声纳在四百米外就捕捉到了反射。他们等着。地面上安排了等着。

吴暗转身看你。他的脸现在在光线里了——四十多岁，颧骨高，眼睛不大，左眼旁边有一道没愈合的划伤。他的嘴张开了，想说什么。

你从他身边走过去。

你不会忘记。但你已经选好了——沉默，比说话重。

"卡德摩斯站事故原因：不可预见的核心谐振。"报告是这么写的。六个人被追授勋章。家属拿到了赔偿金。实验被正式中止。

没有人在报告中提到吴暗。没有人在报告中提到"共振计划"。

你偶尔回到那个入口。站在消光场的边界上。里面没有声音。但你知道他还活着。他在等。

也许有一天第八个人会听到他的脉冲。

也许不会。

---

**选择B：揭露真相**（把他锁在里面）

你没有动拉杆。

吴暗等了五秒。"你在等什么？"

"等你算错。"

你转身就走。他没有反应过来。你按下了墙上的紧急封锁按钮——铁门在你和他之间落下之前，他喊了一声你的名字。不像命令。像在求。

控制室外面。你还有十五秒。你用十秒把郑振的数据、林先和的日志、苏无影的录音打包发送——外骨骼的广播模块能到地面。只要有人在听。

最后五秒。刹车拉杆。你的手掌按在识别面板上。核心停止。

吴暗在铁门后面。敲了三下。停了。可能明白了。

往上爬的路很长。你走得很慢。每到一个地方，你都在对应的人停留过的地方站一会儿。林先和控制室。苏无影的塌方通道。赵铁山的最后一面墙。陈歌的最后一个中继器。郑振的实验室。

六个人。你在心里数——不是名单，是六种方式，你可以选择不坚持。

但你没有。

出口。天是暗的。早上四点。

你坐在应急通道的入口台阶上，等日出。外骨骼的电池还剩三格。

调查持续了两年。承包商破产了。负责人被起诉。六个人的家属拿到了赔偿金。吴暗被判了——你的录音和碎片全部作为证据提交。

你偶尔回到入口。你知道他还活着。你知道他在等。

但你不会再下去了。

---

**选择C：炸毁设施**（全碎片解锁）

你站着没动。吴暗盯着你。

"你不信任我？"

"我在想。"你说，"郑振有个数据我一直没想通。核心地基不稳定。主共振长期满功率运行会让岩层微观断裂。现在过了三天——应该快到临界了。"

他愣住了。

"赵铁山是矿工。"你说。"他下来的时候带着矿用炸药——日常工具，在他那行很正常。我在维护竖井里找到了。我没动它。"

"你想干什么？"

"我在想——如果核心关了，地面上的人拿到了实验数据，换了个地方继续。换个名字。换个设施。六个人的死就是他们的代价——已经付过的代价。"

"所以你要——"

"我要让这个设施不存在。"

你没有等他回答。你转身往回走——不是出口的方向。是竖井的方向。

赵铁山的炸药。你在竖井里找到了他的尸体。他死的时候，炸药就在手边。他没用——他不是放弃的人。他只是不知道往哪炸。

你知道。

计时器设好了。十分钟。你往上爬。每一步都在数——够不够。到底够不够。

第九分钟的时候你还在管道里。最后三十秒——你看到光了。不是天亮——是你的头灯。应急通道的入口就在二十米前。

爆炸时你在离地面十五米的地方。冲击波把你抛到草地上。

你仰面朝天。看着黑色的烟从竖井上升起。核心沉默。设施坍塌。

六个人的名字出现在你的外骨骼屏幕上。一个一个亮起。然后一个一个熄灭。

三年后，地质探测队在废墟中记录到一个新的脉冲信号。它不像人。它像在学习。科学家说那只是地下的空洞回声。

你知道不是。

但你不会再下去了。

## 三、碎片文本重写（第一人称·人物语气）

碎片不是系统日志。碎片是**那个人在那一刻写的东西**。每一块碎片有自己的声音。

### 林先和的碎片

**碎片01**（第1关）："核心频率在自行爬升。我手动降了三次。每次三十秒内它就回到原值。我问是不是有人在远程操作。中央控制室没有回复。我不确定是通讯断了，还是他们不想回复。"

**碎片02**（第3关）："我翻了两天的日志。不是算法。不是故障。有人从内部终端远程登录了核心控制系统。物理来源找到了——在设施内部。还在里面。我没有告诉任何人我在查。我知道这里有第七个人。"

**碎片03**（第6关）："账户名：WU_AN。如果你是我没能等到的第八个人——带上这个。他还在里面。"

### 苏无影的碎片

**碎片01**（第2关）："B区清空了。三个实习生没来得及跑到集合点。我去接他们，通道塌了。我把他们带到C-7走廊——这里氧气还有四小时。我要去找另一条路。"

**碎片02**（第7关）："我找到了第七个人。他说他是来检查实验进度的。我不认识他。他有武器。他杀了小陈——就在我面前。我把他锁在了E-3仓库。他跑不出来。但我也出不去了。"

## 四、角色语音柱

### 主角（玩家）
- **词汇**：技术性的但不卖弄——"液压管"、"消光场"、"暗水"都是你工作的术语，不是解释给外人听的
- **节奏**：短句。多数时候一个想法一行。你不是在讲故事，你是在跟自己确认。愤怒的时候句子会变长
- **你从不说的**：自怜。"他们怎么能这样"——不会。你见过比这更糟糕的事。"我该怎么办"——不会。你选了就选了
- **语气**：疲惫，但没有垮。见过太多事的人的语气。偶尔有黑色幽默——"十年前我们加这个锁，是为了防止有人误入。现在它锁住了六个人退出来的路。"这不是抱怨，这是陈述

### 吴暗
- **词汇**：公事公办的，后勤式。"按要求"、"已确认"、"授权人"——对他而言一切都是流程。杀人也是流程
- **节奏**：开始很稳。最后碎了
- **你从不说的**：道歉。"对不起"——他没说过。他要说的可能是"你不明白"
- **语气**：防御性的。他在说服自己比说服你多
