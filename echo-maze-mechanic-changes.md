# Echo Maze 机制变更清单（v4.1 → v5.0）

对照基础：`echo-maze-knowledge-base.md`（v4.1）
对照目标：v3.1叙事 + 双重脉冲战斗系统 + 10关关卡设计

每个条目标注：保持 / 修改 / 新增 / 删除

---

## 一、导航（保持 1 / 修改 3 / 删除 1）

| 机制 | 变更 | 说明 |
|------|------|------|
| 玩家形象 | **修改** | 旧：先知=兜帽法杖/幽灵=半透明灵体。新：穿声纳外骨骼的工程师（头盔+天线+工业服），在不同角色间切换时外骨骼型号和颜色变化 |
| E出口 | **修改** | 旧：E标识出口。新：闸门/电梯门，视觉上从绿色E变成金属闸门轮廓。解锁后会亮起蓝色轮廓（"通道已打开"） |
| 墙体 | **保持** | 脉冲揭示后渐隐，霓虹边缘发光。10关的墙体纹理需要按关卡主题变换（第1关工业墙板/第8关岩洞墙） |
| 钥匙 | **修改** | 旧：每关一枚钥匙。新：通行权限卡（工业感），视觉从"像素钥匙"改为"像素卡片"。第1关特殊：钥匙替换为"备用电池"，需携带到闸门面板 |
| 玩家角色系统 | **删除** | 旧版"六种回声行者人格"全部推翻，替换为下文"角色系统重制" |

## 三、特殊房间（保持 2 / 修改 2）

| 机制 | 变更 | 说明 |
|------|------|------|
| 共鸣室 | **保持** | 脉冲范围×2。用于第8关（深层岩洞），因为那个关卡鼓励被动声源探测。出现在水晶簇和回音壁的交叉区域 |
| 寂静区 | **修改** | 旧：脉冲无效。新规则不变，但增加限制：寂静区内聚焦脉冲也无效，且不能蓄力（按住空格无反应）。用于第7关（档案室） |

| 秘密房间 | **保持** | 同一墙壁连ping两次揭示。用于隐藏碎片和水晶。在第8关出现 |

## 四、环境元素（全部修改，因为需要兼容聚焦脉冲）

| 机制 | 变更 | 说明 |
|------|------|------|
| 瓦斯云 | **修改** | 旧规则：声波触发扩散，瓦斯+水晶=全屏闪+晕眩捕食者。新规则：探测脉冲触发扩散，聚焦脉冲穿透瓦斯云并点燃——如果聚焦脉冲穿过瓦斯云命中水晶簇，会引发更大的全屏闪光（范围翻倍，但只对瓦斯云覆盖范围内的敌人有效）。用于第2关 |
| 水晶簇 | **修改** | 旧规则：声波折射4方向次级回声。新规则：探测脉冲触发折射（4方向），聚焦脉冲击中水晶簇会**摧毁它**（一次性，不留折射残效）+ 产生短暂全屏闪光。此外新增：**捕食者的巡逻嗡鸣声（每5秒一次）也会触发水晶簇折射**——这是第8关被动声源机制的核心。用于第2、8关 |
| 暗水 | **修改** | 旧规则：能量恢复×3。新规则：暗水中探测脉冲和聚焦脉冲都无效（声波被水吸收），但站立在暗水中每5秒自动恢复2能量。在暗水中移动时，玩家完全看不见——必须凭记忆走。用于第4关 |
| 回音壁 | **修改** | 旧规则：声波反弹显示背后地形。新规则：探测脉冲反弹显示路径地形（保持），聚焦脉冲反弹后方向随机化（因为能量太强，反弹不可预测——不推荐在回音壁附近用聚焦）。用于第8关 |

## 五、敌人（删除 2 / 修改 3 / 新增 2）

| 机制 | 变更 | 说明 |
|------|------|------|
| 捕食者（游荡） | **修改** | 旧：随机走动。新：固定巡逻路线循环。视觉从"生物"改为"无人机"——灰白像素外壳，旋转天线。被探测脉冲惊醒后天线转红。游荡状态下被动嗡鸣声（每5秒一次）可触发水晶簇折射（第8关） |
| 捕食者（狩猎） | **修改** | 旧：听到脉冲冲来，红色闪烁。新：被探测脉冲惊醒后变红，向玩家位置冲刺。聚焦脉冲**不能杀死**普通捕食者——它们是反声能安保无人机，对高强度脉冲有免疫。击中时会被弹飞+晕眩5秒。玩家接触猎人=失去3能量+被弹飞 |
| 捕食者（晕眩） | **删除** | 旧：刚攻击完，无害。新：晕眩是一个可叠加的**状态**而不是独立的敌人类型。任何捕食者被瓦斯爆炸波及或被聚焦脉冲弹飞后进入晕眩状态（黄色闪烁，持续3-5秒）。晕眩结束后恢复巡逻 |
| 捕食者（故障） | **新增** | 用于第8关。固定巡逻路线，不追踪探测脉冲。但追踪**脚步声**——玩家每一步有10%概率吸引最近一只的注意2秒。视觉：外壳破损，天线断裂，间歇性闪烁 |
| 巨兽（休眠） | **新增** | 用于第7关。蜷缩在档案室中央，比正常无人机大三倍。不移动。每隔数秒脉动一次红光。玩家步数超过50后醒来——醒来后产生以自身为中心扩散的红色声场（扩张速度=1格/秒），触碰即被弹飞到关入口。不可击败。只能通过控制步数来规避 |

## 六、Boss（全部重写）

| 机制 | 变更 | 说明 |
|------|------|------|
| 回响之母 | **修改** | 旧：近身ping3次击败，你ping她也ping。新：**3层护盾**。探测脉冲找它位置（但被它反ping干扰地图），**必须在3格内发射聚焦脉冲**才能破盾。每次击穿1层→它弹飞+发射3次不规则反ping→再次接近→重复。出现频率从3次改为1次（第3关）。视觉：深蓝圆盘，表面脉冲波纹扩散 |
| 静默猎手 | **修改** | 旧：偷吃能量水晶，ping3次击败。新：**3层护盾**。它抢房间里的水晶每吃1个加1层额外护盾。你必须在它没额外护盾时发射聚焦脉冲。每次击中后房间4只无人机冲向你。第3次击中时它移动速度最快。出现频率从3次改为1次（第6关）。视觉：黑色六边形机体，6个传感器阵列 |
| 虚空化身 | **修改** | 旧：制造假墙壁，ping3次击败。新：**3层护盾**。制造假墙+隐藏真墙+间歇性反噬波（你ping的同时它也ping→2秒全暗）。聚焦脉冲只能在**频率锚点**上发射才能伤害它。3个锚点散布在房间中，每次击中后锚点位置随机重排。3个锚点激活后，第4个锚点在它脚下出现——最后一击。出现频率从3次改为1次（第9关）。视觉：由密度波纹构成的人形，在六人间切换外形 |

### Boss统一规则
- 所有Boss：3层护盾。不用血条。
- 伤害来源：聚焦脉冲（长按空格>0.5秒释放）。探测脉冲不造成伤害。
- Boss掉落：每次护盾碎裂，掉落1-2个能量水晶（确保玩家有能量打下一层）
- 失败条件：能量耗尽且无可收集水晶 → 自动每秒恢复0.5能量（仅够探测脉冲，不够蓄力到聚焦），软锁保护

## 七、角色系统（全部重写）

旧版六种角色基于"不同回声行者"的抽象概念设计。新版对应六个人的外骨骼型号，玩家在特定条件达成后解锁。

| 角色名 | 对应人物 | 外骨骼特性 | 解锁条件 | 设计理由 |
|--------|---------|-----------|---------|---------|
| **先知** (林先和的遗物) | 林先和 | 脉冲范围+3格 / 能量上限-2（从10降到8） | 第3关击败回响之母后自动解锁 | 他过度分析一切，代价是消耗更快。视野大但续航短 |
| **幽灵** (苏无影的外骨骼) | 苏无影 | 捕食者不会追踪你的探测脉冲 / 脉冲范围-40% | 第7关完成后解锁 | 安保外骨骼，低调运行。用更窄的视野换取安全 |
| **巨兽** (赵铁山的改装) | 赵铁山 | 撞墙时墙面被撞碎（仅限特定脆墙标记）/ 每走一步消耗0.3能量 | 第4关找到赵铁山的尸体后解锁 | 他的笨办法——暴力破墙。每一格都耗能，但可以走捷径 |
| **歌者** (陈歌的中继模式) | 陈歌 | 探测脉冲免费 / 范围缩小到2格 / 不能使用聚焦脉冲 | 第5关完成后解锁 | 她放弃了能力以求生存。你获得无限次脉冲，但每次只能看见身边两格 |
| **暗影** (吴暗的装备) | 吴暗 | 移动不留下声学痕迹（捕食者听不到脚步声） / 发射脉冲后你看不到自己的位置（小地图上你的标记消失3秒） | 结局A后解锁 | 他抹除一切痕迹——包括他自己。你变得不可追踪，但也看不见自己 |
| **共振者** (郑振的感应器) | 郑振 | 静止3秒后自动揭示周围5格（被动） / 不能主动发射任何脉冲（空格完全禁用） | 结局B后解锁 | 她只观察，不干涉。你学会了不动——世界会自己显现 |

### 默认解锁
- 标准外骨骼（玩家默认）：无任何加成，标准参数。能量10，脉冲范围7格。

## 八、词缀（保持 2 / 删除 4）

词缀是关卡的难度修正选项。旧版有6个。新版保留2个核心词缀，删除4个不相关的。

| 词缀 | 变更 | 说明 |
|------|------|------|
| SILENT | **保持** | 禁用脉冲（探测+聚焦都禁用）。分数+200%。叙事：外骨骼声纳模块故障 |
| HUNTED | **保持** | 捕食者数量×3。分数+150%。叙事：安保系统全面激活 |
| BLIND | **删除** | 出口隐藏。与叙事冲突——出口是闸门，物理存在 |
| FRAGILE | **删除** | 能量减半。与角色"先知"特性重叠 |
| AMNESIA | **删除** | 墙壁记忆-50%。太依赖已有视觉系统 |
| STORM | **删除** | 每30步自动脉冲。与第5关中继器机制冲突 |

## 九、操作（修改1处）

| 操作 | 变更 | 说明 |
|------|------|------|
| 空格（脉冲） | **修改** | 旧：按一次=发射脉冲。新：**轻按**（<0.2秒）=探测脉冲，消耗1能量 / **长按**（>0.5秒后释放）=聚焦脉冲，消耗3能量。长按过程中屏幕边缘出现蓝色收缩环（视觉提示），0.5秒后环闭合——此时释放即发射聚焦脉冲。提前释放降级为探测脉冲 |
| 移动 | **保持** | WASD/方向键 |
| 手机 | **保持** | 滑动移动 / 点按=探测脉冲 / 长按>0.5秒=聚焦脉冲 |
