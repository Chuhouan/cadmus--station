# CADMUS STATION v0.001

> 你看不见迷宫。你只能听见它的回声。

卡德摩斯地下声场研究站的主共振核心过载。六个工作人员失联。你穿着旧型号声纳外骨骼爬下去——不依赖光线，只靠超声波回声在完全黑暗中导航。

你的任务是关闭核心，找到幸存者，活着爬出来。但你很快会发现——这不是事故。

---

## 运行

```bash
pip install flask flask-socketio
python server.py
# 打开 http://localhost:5000
```

## 操作

| 按键 | 功能 |
|------|------|
| WASD | 移动 |
| 空格（轻按） | 探测脉冲——消耗1能量，7格范围，揭示墙壁 |
| 空格（长按0.5秒） | 聚焦脉冲——消耗3能量，前方3格扇形，破Boss护盾 |
| TAB | 打开叙事碎片面板 |
| H | 系统手册 |

## 技术栈

Python + Flask + Socket.IO + Canvas 2D 像素渲染。零 emoji，全部手绘像素图案。
